> A simple typescript library starter using rollup.

## Getting started.
1. Prerequisites
    - [NodeJs LTS](https://nodejs.org/zh-cn/)
    - [Yarn LTS](https://classic.yarnpkg.com/zh-Hans/docs/install)

2. Install dependencies
    ```bash
    yarn
    ```
2. Development
    ```bash
    yarn dev
    ```

2. Build
    ```bash
    yarn build
    ```
